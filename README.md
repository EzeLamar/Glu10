# Proyecto IAP

Proyecto de la materia Ingeniería de Aplicaciones.



Este proyecto estará dividido en 2 partes:

1. **FrontEnd**:
   - Angular 6
   - API Google Maps
   - Stylesheet: Material
   - Json-server
   - Auth0 API
2. **Backend:**
   - REST API PHP7
   - MySQL
   - Apache
3. **Otros**:
   - Github
   - Dia
   - Figma
